Implementing Delete Selection in Wakanda
This is the companion project for the technical note of the same name. You can select one or more Companies and click the Delete Selection button to remove them.
To implement a Delete Selection function for our application we have written a collection method on the Company dataclass and made it public so we can call it when the Delete Selection button is clicked. We pass the selected rows to our server method. It removes any selected company that does not have employees and returns the updated company collection. The companies that could not be deleted are then highlighted.
